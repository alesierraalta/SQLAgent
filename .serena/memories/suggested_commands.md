- Crear entorno virtual: `python -m venv .venv`
- Activar entorno virtual (Windows): `.venv\Scripts\activate`
- Instalar dependencias: `pip install -r requirements.txt`
- Copiar variables de entorno ejemplo: `copy .env.example .env`
- Generar datos de prueba: `python scripts/generate_test_data.py`
- Ejecutar CLI (query): `python -m src.cli query "<pregunta>" --limit 10 --verbose`
- Ver schema: `python -m src.cli schema`
- Probar conexión: `python -m src.cli test-connection`
- Validar SQL manualmente: `python -m src.cli validate-sql "<sql>"`
- Ver estadísticas: `python -m src.cli stats --days 7 --slow-threshold 5`
- Ejecutar tests: `pytest`
- Tests con coverage: `pytest --cov=src --cov-report=html`
- (Opcional) Smoke DB local: `python scripts/generate_test_data.py` con `DATABASE_URL` apuntando a Postgres local (usuario read-only recomendado).
